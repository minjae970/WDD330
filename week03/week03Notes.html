Week02 Notes

  • Object methods, "this"

  • To create an object function: object.(method) = function() { (insert code here); }; You can pre-declare this function by doing function method() { (insert code here); }; and object.method = method;

  • Shorthand syntax for methods in an object is object = { method() { code;}}; instead of doing the method: function() { code;}.

  • “this” refers to the current object.

  • “this” is not bound as “this” is evaluated at run time.

  • Unbound “this” allows functions to be reused for different objects, but creates more potential errors and mistakes.

  • Arrow function uses “this” from the outer context.

  • Chapter 5: Objects

  • If an object’s property is a function, it is called a method.

  • Objects can inherit properties from other objects.

  • Object literal is object that is created directly in the language by wrapping all its properties in curly braces. This allows creation of objects without defining classes.

  • Using constructor function allows us to create empty objects.

  • You access properties with a period or bracket.

  • You can call methods with period or bracket.

  • You can check if a property exists by using “in” or hasOwnProperty function

  • You can add or change properties to objects at any time.

  • You can delete properties with “delete”

  • Objects are copied by reference.

  • JSON is string representation of object literals.

  • Math object is a built in object for constants and methods.

  • Date object contains information about date and time. Each object represents a single moment in time.

  • The RegExp object is used to search strings for matches to the pattern.

  • Chapter 6: The Document Object Model

  • DOM represents an HTML document as a network of connected nodes that forms tree-like structure.

  • All nodes have numerical code to signify type. 1 is element, 2 is attribute, 3 is text, 8 is comment, and 9 is body.

  • Document.querySelector() method allows you to use CSS notation to find first element in the document that matches a CSS selector provided as argument. querySelectorAll() method returns a node list of all that matches the CSS selector argument.

  • We can find the text value of a node using node.nodeValue or variable.textContent.

  • We can get or set attributes using getAttribute() and setAttribute() methods.

  • We can set the class of an element by using className.

  • createElement(), createTextNode(), appendChild() allows us to create markup dynamically.

  • You can update css of an element by using element.style property

  • Chapter 7: Events

  • addEventListener() method allows us to add an event listener that does something when some condition is met.

  • You can have inline event listeners.

  • “target” returns a reference to the node that fired the event.

  • There are mouse events (click, mouse down, mouse up, dblclick, mouseover, mousemove) and keyboard events (keydown, keyup, keypress).

  • There are touch events for modern devices (touchstart, touchend, touchmove, touchenter, touchleave, touchcancel).

  • Eventlisteners can be removed by using removeEventListener.

  • You can prevent default behavior by using preventDefault() method.
